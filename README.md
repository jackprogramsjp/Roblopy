# Roblopy
Roblox API built in Python.

To install Roblopy:
```
pip install roblopy
```

How to import?
```
from roblopy.api.[name] import [class]
```

For example:
```
from roblopy.api.user import Users
```

**DO NOT IMPORT CLASSES WITHOUT S; FOR EXAMPLE, DO NOT IMPORT USER. IMPORT USERS WITH AN S.**
